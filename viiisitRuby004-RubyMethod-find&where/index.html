怎麼用 Ruby 語法查詢資料？

  • find vs find_by vs where

    • find 方法使用 key 值來查找，返回該記錄的物件或拋出 exception (如果找不到對應的記錄，則會拋出 ActiveRecord::RecordNotFound 錯誤。)例如：Model.find(id)。

    • find_by 方法使用 key & value 的 Hash 來查找記錄，返回符合條件的第一個記錄的物件或 nil。例如：Model.find_by(column_name: value)。

    • where 方法接收一個條件 Hash 或 SQL 字符串作為參數，並返回符合條件的所有記錄。例如：Model.where(column_name: value) 或 Model.where(‘column_name = ?’, value)。
      返回結果：where 方法返回一個 ActiveRecord 查詢集合（Relation），可以進一步應用其他方法（如 order、limit、group 等）或迭代處理查詢結果。

    需要注意的是，find_by 方法只返回符合條件的第一個記錄，如果需要返回多個記錄，可以使用其他方法如 where；where 方法返回的是一個 ActiveRecord 查詢集合，而非單個物件。可以使用其他方法（如 first、last、take 等）從查詢集合中獲取單一記錄。總結而言，where 方法用於根據條件查詢資料庫記錄，返回符合條件的記錄集合。
    此外，find 方法使用主鍵查找記錄的效能較高，而 find_by 方法則更靈活，可以根據多個欄位進行查詢。
