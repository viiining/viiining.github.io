Ruby 簡介

  • 發明人：松本行弘　（まつもとゆきひろ）
  • Object-Oriented Programming (OOP) 物件導向程式語言
    • 在 Ruby 裡，萬物皆物件
    • 所有的資訊與程式碼，都給傳遞給物件擁有的屬性（properties）與行為（actions）
    • 屬性為實體變數（instance variables）、行為稱為方法（methods）
  • Ruby on Rails 是一款使用 Ruby 程式語言所開發出來的網站開發框架

Environment Setting (MacOs)

  • 下載 HomeBrew
    • 在 Terminal 執行以下指令，來安裝 Ruby
  • 下載 RVM 來管理 Ruby 版本
