Ruby on Rails - Practicing CRUD

CRUD

CRUD 是 Create, Read, Update, Delete 四個字母的縮寫，中文是「增刪查改」；
這是一連串常見的動作行為，通常是為了針對某個特定資源所作出的舉動（例如：建立資料、讀取資料等）。
這四個行為最常見的用途是在使用 SQL 資料庫與網站的 API 埠口的時候。

  • 基本用途與別名
- SQL HTTP REST 表現層狀態轉換
Create 建立 INSERT PUT / POST POST
Read 讀取 SELECT GET GET
Update 更新 UPDATE PUT / POST / PATCH PUT
Delete 刪除 DELETE DELETE DELETE

實作一個網站，這網站有 homepage, aboutpage…，有了想要新增的頁面時，
依序由 Route -> Controller -> View 來建立：

  • 先在 routes.rb 檔案裡建立新路徑
  • 接著在 pages_controller.rb 檔案裡建立 action
  • 接著在 pages 資料夾裡，手動新增檔案：action 名稱.html.erb，建立 view

在 Rails 中，POST 行為具有保護機制。
